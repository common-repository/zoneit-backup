Create backup from website files and db

== Description ==

oPing backup is the plugin for creating backup from your website. This plugin is creating the backup from all files of website and database (.sql). it has been developed by <a href="https://oping.cloud">oPing Cloud</a>.

You Can use this plugin for migrating your website to oPing Cloud.

Features:
- Create Backup - manually
- View And Delete Backups List
- Restore Backup

== Installation ==

1. Upload the plugin files to the `wp-content/plugins/zoneit-backup` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings-oPing backup screen to configure the plugin

== Frequently Asked Questions ==

= How I use this plugin =

You can go to Settings -> oPing Backup and then click on create backup. The backup process starts. The backup process may take some time. By refreshing this page, you will see the backup link.

= This plugin is free =

Yes, it is free and no premium account is needed.
